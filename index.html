RWF Sidebar Prototype

This aims to be proof of concept for a fully functional, production-ready sidebar navigation system built with pure HTML, CSS, and vanilla JavaScript.

Features include:

• Minimizable sidebar with smooth transitions
• Smart search with keyboard navigation
• Hierarchical L1/L2/L3 navigation
• Proper accessibility implementation
• Modern hover/focus states
• Clean, professional styling

Features missing :

• Favouriting control
• Active sidebar states and corresponding pages

Built in ~5 hours using Cursor AI. Ready for dev team feedback. 🚀
